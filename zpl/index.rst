Welcome to ZenPack Library
=============================================

The ZenPackLib allows you to automate much of the ZenPack creation process.
It does this by:

* Automating boilerplate code into dynamically created classes and modules
* Automating certain Zenpack components like the GUI and ZCML
* Defining standard classes and modules needed for Zenpack modification
